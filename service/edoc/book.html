This module provides a common service for the Book Recommendation system.

Version: 0.0.1

Behaviours: cloudi_service.

Authors: Bruce Kissinger.

Description

This module provides a common service for the Book Recommendation system. It leverages the CloudI framework to provide a cloud-based service that can be used by many different programming languages.

The exported functions are required interfaces that the service must implement.
